This program provides a straightforward, easy-to-set-up way to press buttons in BitFocus Companion by listening to MIDI note-on messages from software such as Proclaim or ProPresenter.

  • Prerequisites: make sure there is a suitable MIDI port available for this program to listen on. On Windows, install loopMIDI and create a loopback MIDI port
  • Install and configure this program in its settings tab, including selecting the MIDI port to listen on
  • Send MIDI notes from Proclaim, ProPresenter or other software of your choice to the same MIDI port, with:
    • MIDI channel = Companion page number
    • Note = Companion button row number
    • Velocity = Companion button column number
    For example, to press Companion button 1/0/2, send a MIDI note-on with channel=1, note=0, velocity=2
  • If you need to acccess a Companion button beyond page 16, use the Page offset setting
